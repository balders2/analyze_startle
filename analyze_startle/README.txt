
Installation
1. add C:\Python27 to path (See: https://www.java.com/en/download/help/path.xml)
2. add C:\Python27\Scripts to path
3. run install.bat
4. follow prompts.

To run
1. copy tools to directory with psychophysiology files (.acq)
	a. analyze.exe
	b. analyze.py
	c. run_analysis.bat
2. double-click run_analysis.bat to run
